## 一、产品定位

**Banana Pro / Banana 2** 是一款面向**软装设计师**的 AI 设计解析与可视化工具。

核心定位：**"我给AI的 → 一分钟后AI给我的"** —— 设计师上传原始素材（SU截图/平面图/毛坯房照片/室内照片），AI快速输出专业级设计解析图纸与效果图。

## 三、核心功能模块详细拆解

### 模块一：渲染效果增强（SU → 写实摄影）

**场景**：设计师有SU基础模型截图，需要转为专业室内摄影效果图。

**输入**：SU场景截图（含家具、硬装、软装布局）

**AI处理要点**：
- **几何保真**：严格保持空间结构、物体位置、墙体轮廓、门窗尺寸
- **物体完整性**：不新增、不删除、不简化任何家具/装饰
- **材质转换**：SU基础面 → PBR物理材质（反射、粗糙度、法线、金属度、凹凸）
- **光影系统**：
  - 环境光：柔和自然天光从窗外射入，高光自然曝光
  - 室内光：人工暖光源 3000K-3500K，多层次立体照明
- **摄影参数**：尼康Z9、光圈f/8、快门1s、ISO 64、24-70mm镜头、HDR包围曝光
- **可选人物**：室内添加1个人物，长曝光形成虚焦

**输出**：8K超高清写实室内摄影图

**参考提示词文件**：`渲染效果展示.jpg`

### 模块二：多风格室内彩平图

**场景**：设计师有线稿/黑白平面户型图，需要快速生成各种风格的彩色平面图。

**输入**：原始户型线稿图（需保持结构不变）

**支持风格**（从效果图中提取）：

| 风格名称 | 视觉特征 |
|---------|---------|
| **手绘质感彩平** | 橙色粗体墙线、米色/浅木色地面、蓝白方格卫浴、绿植点缀、手写体英文标注、粗糙纸张纹理 |
| **马克笔风格** | 绿色墙线、浅蓝/浅灰分区底色、橙色蓝色手绘线条、专业建筑图纸质感 |
| **中古风** | 暖棕/奶咖底色、深棕木地板、橙色沙发、复古绿卧室、格纹图案 |
| **新中式禅意** | 水墨竹影背景、宣纸质感、户型主体清晰、内部纹理极淡、弱对比氛围 |
| **治愈系卡通** | 暖橙色墙线、高饱和分区色彩、深棕木纹+黑色地面、彩色家具剪影、白色小人 |
| **奶油风** | 暖米色/奶油色基调、浅木色家具、整体柔和温馨 |
| **深灰大理石风** | 客厅深灰大理石、厨卫浅灰大理石、浅棕斜拼木地板、白色家具 |
| **2D扁平极简** | 浅灰底色、淡显黑色竖向线条、浅灰蓝色厨卫填充、白色平面活动线稿人 |

**核心约束**：
- 严格保留原始户型结构与线稿
- 无透视、无轴测、无变形
- 比例精准
- 墙体/门窗保持样式不变

**参考效果**：`提示词及效果展示 (1)-(4), (14)-(18)`

### 模块三：家具材质分析图

**场景**：分析室内空间中各家具的材质构成与性能。

**输入**：室内空间照片

**输出结构**：
1. **原图材质指引**：在原图上用引线标注各家具材质
2. **分模块展示**：每个模块搭配"材质纹理图标 + 特性说明"
3. **材质性能分析表**：
   - 列：材质类型（真皮/大理石/金属/羊毛等）
   - 行：耐用性、维护难度、质感表现、环保等级

**视觉风格**：专业住宅室内分析示意图，线条利落精准，信息层级清晰，中英文标注

**参考效果**：`提示词及效果展示 (5)`

- 确保空间结构、氛围、布局不变
- 要有故事性和连续性

### 模块六：室内空间尺度分析图

**场景**：为户型图添加专业尺度分析与人体工学示意。

**输入**：室内平面布局图（含家具布置）

**输出结构**：
1. **尺度标注**：平面图各区域边缘添加精确尺寸数字 + 尺寸线
2. **人体姿态简图**：在关键区域（厨房、卫生间、客厅）放置标准化人体姿态（站立、坐姿、弯腰操作），叠加标注
3. **家具配置**：各功能区匹配对应家具，标注关键家具尺寸
4. **1:1对比示意图**：下方展示四组"人体模型+家具"的尺度对比

**视觉风格**：专业住宅室内尺度分析示意图，配色与平面相似，线条利落精准，符合住宅室内设计规范，中英文标注

### 模块七：家具智能布局

- 真实材质细节，表面真实反射
- 明亮室内光线，高级摄影感
- 按原图尺寸比例生成

### 模块十：颜色搭配分析图

## 四、通用技术约束与要求

### 4.1 几何保真（贯穿所有模块）
- 严格保持原始空间结构、物体位置、墙体轮廓
- 严禁修改空间布局、物体比例与相对位置
- 完整保留所有家具、灯具、装饰品的原始样式与尺寸比例
- 不新增、不删除、不简化任何物体

### 4.2 图像质量要求
- 8K/4K超高清输出
- 无噪点、无锯齿
- 光影物理准确
- 材质质感细腻
- 空间纵深感强

### 4.3 风格一致性
- 用户指定的风格必须在输出中严格体现
- 材质、色调、氛围需与指定风格匹配

### 4.4 双语标注
- 涉及专业图纸时，需中英文标注
- 家具名称、材质说明等需双语

    ▼

    ├── 风格选择

    │
    ▼
[AI生成] → 进度展示
    │
    ▼
[结果展示]
    ├── Before/After对比
    ├── 多方案选择

    │
    ▼
[下载/导出]
    ├── 高清图片

## 六、与前代产品（Banana 2）的关系

从截图中的品牌标识来看，产品经历了迭代：
- **Banana 2**：室内应用汇总阶段，侧重功能演示
- **Banana Pro**：升级版本，强调"一分钟出图"的高效体验

Pro版本应在前代基础上：
1. **优化交互**：更简洁的上传-选择-生成流程
2. **模板化**：将各种风格/输出类型固化为可一键选择的模板
3. **批量能力**：支持同时处理多个场景/多种风格
4. **项目管理**：设计师可以为不同客户创建项目，保存历史方案
